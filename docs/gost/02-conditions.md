# 2. Условия выполнения программы

## 2.1. Требования к техническим средствам

Для функционирования Программы необходим ПЭВМ со следующими характеристиками:

* Процессор: с тактовой частотой не менее 1 ГГц.

* Оперативная память: не менее 512 Мб.

* Свободное место на диске: не менее 100 Мб.

## 2.2. Требования к программным средствам

Для работы Программы необходимо наличие установленного программного обеспечения:

* Операционная система: Windows 10/11, Linux или macOS.

* Платформа: .NET Runtime версии 8.0 и выше.
