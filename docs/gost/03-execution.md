# 3. Выполнение программы

## 3.1. Загрузка и запуск

1. Загрузить библиотеку `ClinicCore.dll` в директорию приложения.

2. Инициализировать класс `Patient` в вызывающем коде.

## 3.2. Порядок работы

Для расчета возраста необходимо:

1. Задать свойство `BirthDate`.

2. Вызвать метод `CalculateAge()`.

Пример использования:

```csharp

var p = new Patient { BirthDate = DateTime.Parse("1990-01-01") };

int age = p.CalculateAge();
