# 4. Сообщения оператору

В процессе работы Программы могут возникать следующие аварийные ситуации:

| Сообщение / Исключение | Причина возникновения | Действия оператора (программиста) |

| :--- | :--- | :--- |

| `DivideByZeroException` | Значение роста (`Height`) равно 0 или отрицательное. | Проверить входные данные перед вызовом метода `CalculateBMI`. Установить корректный рост. |

| `System.NullReferenceException` | Объект класса `Patient` не инициализирован. | Создать экземпляр класса через конструктор `new`. |
