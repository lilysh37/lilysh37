# 1. Назначение программы

## 1.1. Назначение

Программа **ClinicCore** (далее — Программа) предназначена для автоматизации процессов расчета биометрических показателей пациентов в медицинских учреждениях.

## 1.2. Функциональные возможности

Программа обеспечивает выполнение следующих функций:

1. Хранение и обработка персональных данных пациента (ФИО, Дата рождения).

2. Вычисление полного возраста пациента на текущую дату.

3. Расчет индекса массы тела (ИМТ) на основании веса и роста.
